# Manual do Usuário - Gerenciador de Disciplinas

## Funcionalidades

O Gerenciador de Disciplinas oferece as seguintes funcionalidades:

1. Fazer Inserção: Permite cadastrar cursos, disciplinas, professores e associações de disciplinas.
2. Fazer Impressão: Permite imprimir informações sobre cursos, disciplinas, professores e associações de disciplinas.
3. Fazer Remoção de uma Associação de Disciplina: Remove uma associação de disciplina com base no código da disciplina e no ano letivo.
4. Carregar Arquivo em Lote: Permite carregar informações de um arquivo de texto em lote.

## Utilização

### Menu Principal

No menu principal, você encontrará as seguintes opções:

1. Fazer Insercao: Opção para realizar inserções de informações.
2. Fazer Impressao: Opção para imprimir informações.
3. Fazer Remocao de uma Associacao de Disciplina: Opção para remover uma associação de disciplina.
4. Carregar Arquivo em Lote: Opção para carregar informações de um arquivo de texto.
5. Finalizar o Programa: Opção para sair do programa.

### Cadastros

Dentro da opção "Fazer Insercao", você poderá realizar os seguintes cadastros:

1. Cadastrar Curso: Permite cadastrar um novo curso.
2. Cadastrar Disciplina: Permite cadastrar uma nova disciplina.
3. Cadastrar Professor: Permite cadastrar um novo professor.
4. Cadastrar Associacao de Disciplina: Permite cadastrar uma nova associação de disciplina.

### Impressões

Dentro da opção "Fazer Impressao", você poderá imprimir informações sobre:

1. Cursos
2. Disciplinas
3. Professores
4. Associações de Disciplinas por níveis
5. Associações de Disciplinas in-ordem
6. Registros livres

### Remoção de Associação de Disciplina

Ao escolher a opção "Fazer Remocao de uma Associacao de Disciplina", você será solicitado a fornecer o código da disciplina e o ano letivo da associação a ser removida.

### Carregamento de Arquivo em Lote

Ao escolher a opção "Carregar Arquivo em Lote", você deverá digitar o nome do arquivo de texto que contém as informações a serem carregadas no sistema.

## Encerramento

Para finalizar o programa, basta selecionar a opção "Finalizar o Programa" no menu principal.
